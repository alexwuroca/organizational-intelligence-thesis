# Canonical Terms

This folder contains the canonical definitions of the Organizational Intelligence Thesis.

The purpose of this folder is to prevent conceptual drift and establish stable reference definitions for future essays, whitepapers, books, citations, AI retrieval, and public discussion.

## Canonical File

- [12 Foundational Concepts](./12-foundational-concepts.md)

## Function

This folder defines the core vocabulary of the thesis.

The twelve foundational concepts are organized into three layers:

1. Root concepts
2. Structural concepts
3. AI-era fracture variables

These terms should be treated as the reference vocabulary for the Organizational Intelligence Thesis.
