# Six-Layer Pressure Test

**Author:** Alex Wu Roca  
**Thesis:** Organizational Intelligence Thesis  
**Version:** v0.1 canonical draft

---

## Purpose

This framework tests claims in the Organizational Intelligence Thesis.

It exists to prevent self-confirming reasoning.

Every major claim should be tested across six layers:

1. Execution layer
2. Competition layer
3. Structural layer
4. Institutional layer
5. Civilizational layer
6. Philosophical layer

---

# 1. Execution Layer

## Question

Can it actually be built?

## Tests

1. Can the concept be translated into operational design?
2. Can a firm identify the relevant organizational intelligence?
3. Can the intelligence asset be documented?
4. Can it be governed by permissions?
5. Can it be audited?
6. Can responsibility be assigned?
7. Can the process survive human error?
8. Can the structure survive legacy systems?

## Failure Mode

The thesis becomes elegant but unusable.

---

# 2. Competition Layer

## Question

Can it survive once built?

## Tests

1. Can competitors copy the system?
2. Can AI vendors capture the intelligence layer?
3. Can platforms absorb the control rights?
4. Can open-source alternatives weaken proprietary advantage?
5. Does the firm retain any durable edge?
6. Does control create value faster than competitors erode it?
7. Can the firm prevent dependency on external systems?

## Failure Mode

The firm builds organizational intelligence but loses control of it to vendors, platforms, or competitors.

---

# 3. Structural Layer

## Question

Is the world model wrong?

## Tests

1. Did firms ever fully contain capability?
2. Is AI truly different from previous software, outsourcing, or consulting?
3. Is organizational intelligence distinct from knowledge management?
4. Are responsibility, authority, and capability truly separable?
5. Does the model explain both small and large organizations?
6. Does it explain high-risk and low-risk domains?
7. Does it work across industries and institutional systems?

## Failure Mode

The thesis describes a real phenomenon but exaggerates its novelty or scope.

---

# 4. Institutional Layer

## Question

Is it legally and regulatorily allowed?

## Tests

1. Can the relevant rights be contractually defined?
2. Can trade secret law protect the relevant intelligence?
3. Can employment agreements address capability sedimentation?
4. Can data protection law permit the required use?
5. Can regulated industries allow AI-mediated invocation?
6. Can liability be assigned clearly?
7. Can audits and records satisfy regulators?
8. Can cross-border restrictions be handled?

## Failure Mode

The theory works strategically but fails under law, compliance, or liability.

---

# 5. Civilizational Layer

## Question

Is it socially stable?

## Tests

1. Does the structure preserve accountability?
2. Does it intensify worker dispossession?
3. Does it centralize control dangerously?
4. Does it undermine trust?
5. Does it create responsibility gaps?
6. Does it increase systemic fragility?
7. Does it create new dependency on opaque systems?
8. Does it preserve human dignity in organizations?

## Failure Mode

The structure is efficient but socially unstable.

---

# 6. Philosophical Layer

## Question

Does it misunderstand the nature of human organization?

## Tests

1. Are firms only capability containers?
2. Are organizations also meaning systems?
3. Are trust, legitimacy, ritual, identity, and status underweighted?
4. Can responsibility exist without human moral agency?
5. Can intelligence be separated from judgment?
6. Can judgment be separated from lived experience?
7. Can coordination be reduced to control?
8. Does the thesis over-systematize human institutions?

## Failure Mode

The thesis becomes technically powerful but philosophically shallow.

---

# Audit Score

Each claim can be rated from 0 to 5 on each layer.

0 = fails completely  
1 = weak  
2 = partially survives  
3 = acceptable  
4 = strong  
5 = very strong

## Passing Rule

A claim should not enter the canonical thesis unless:

1. It scores at least 3 on Execution.
2. It scores at least 3 on Structure.
3. It does not score 0 on Institution.
4. It has an explicit responsibility answer.
5. It does not collapse under the strongest philosophical objection.
