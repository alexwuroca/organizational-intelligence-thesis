# Core Thesis

This folder contains the central thesis of the Organizational Intelligence Thesis.

## Canonical File

- [Core Thesis One Page](./core-thesis-one-page.md)

## Central Question

When organizational capability can be designed, trained, invoked, replicated, and controlled outside traditional firm boundaries, why should the firm remain the default container for complex coordination?

## Function

This folder provides the shortest stable statement of the thesis.

It should be used as the primary reference page for the core argument.
