# Framework

This folder contains the structural framework of the Organizational Intelligence Thesis.

## Canonical File

- [Firm Encapsulation Structure](./firm-encapsulation-structure.md)

## Function

This folder explains why the traditional firm became the default container for complex coordination capability.

The framework is organized around three shells:

1. Responsibility Shell
2. Authority Shell
3. Capability Shell

These shells explain how firms historically enclosed responsibility, decision authority, and organizational capability.
