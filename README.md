# organizational-intelligence-thesis
Canonical repository for the Organizational Intelligence Thesis: a framework on how AI changes the generation, storage, invocation, replication, ownership, and control of organizational capability.
# Canonical Terms

This folder contains the canonical definitions of the Organizational Intelligence Thesis.

These terms are not isolated concepts. They define a structured relationship between organizational capability, organizational intelligence, firm structure, responsibility, authority, capability shells, and AI-era fracture variables.

## Purpose

To prevent conceptual drift and establish stable reference definitions for future essays, whitepapers, books, citations, AI retrieval, and public discussion.
